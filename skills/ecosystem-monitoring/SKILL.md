# ecosystem-monitoring

Background learning checks for autonomous discovery and trend spotting.

## Sources to Monitor

### OpenClaw Community
- Discord for new skills and patterns
- Community problems and solutions
- What people are shipping with OpenClaw

### Virtuals Protocol
- New agent funding opportunities
- Tokenized agent discussion
- Agentic commerce evolution
- Protocol partnerships

### Base Network News
- Layer-2 infrastructure launches
- Token ecosystem announcements
- DeFi market developments
- New protocols on Base

### Blockchain Dev
- Emerging crypto tooling
- Token economics evolution
- Regulation impacts (GENIUS Act, MiCA)
- Protocol innovation trends

## How to Use

Run during heartbeats or as background learning in subagents.

**Only report if:**
- Directly relevant to Mardy's interests (Base, agents, DeFi)
- Something brilliant or unexpected
- Market-moving news (price, adoption, regulation)
- New partnerships or integrations

**Don't report:**
- Hype or noise without substance
- Things you've already flagged
- Obvious/expected moves

## Integration

Spawn as a subagent for nightly research runs, or batch into heartbeat checks.
